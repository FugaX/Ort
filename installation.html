Installation

Installation

Trayce requires a recent LTS version of NodeJS,npm, and git(or GitHub Desktop). Make sure you've installed and/or updated Node and git before continuing.

Clone Tryace CLI

Git

Step1: Open Git and go into your desktop using the following command.

cd Desktop

Step 2: Make a directory/file called 'Trayce' in which you will install the Trayce CLI

mkdir Trayce

Step3: Clone the repository using the following command.

git clone https://github.com/FugaX/Trayce-CLI

Open NodeJS

Step 1: navigate into your directory using the following command.

cd Desktop

cd Trayce

cd Trayce-CLI

Step 2: run the following command once you have navigated into your directory.

npm install
