Quick Start

Want to see Trayce in action? Start by installing and running our starter template!

# Trayce requires a recent LTS version of git.

$ git clone https://github.com/TrayceUI/Trayce-Quick-Start

# Go into the repository.

$ cd Trayce-Starter

# Run index.html to see Trayce in action.
